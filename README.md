Institute of Digital Risk – Assignment

Overview
This project presents a responsive homepage and logo design for the Institute of Digital Risk (IDR). The goal was to create a clean, modern website that communicates IDR’s mission of advancing digital, cyber, and AI risk capabilities through training, innovation, and industry collaboration.

Logo Design Rationale
The logo uses a geometric cube form to symbolize structure, stability, and resilience within complex digital systems. The orange colour represents innovation, proactive thinking, and the dynamic nature of technology risk, while black conveys security, authority, and trust in regulated environments. A clean sans-serif typography was selected to reflect a modern technology and education focused institution. The minimalist design ensures clarity and legibility across different sizes, including mobile headers and favicons.

Website Features
- Responsive single-page homepage
- Sticky navigation bar with smooth scrolling
- Hero section with call-to-action
- About section explaining the institute's mission
- Service pillars (Academy, Innovation, Advisory)
- Pipeline model (Train → Hire → Innovate → Deploy)
- Community section describing target audience
- Contact form for registering interest

Technologies Used
- HTML5 (semantic structure)
- CSS3 (Flexbox and responsive design)
- Vanilla JavaScript (smooth scrolling)

How to Run
Download the repository and open "index.html" in any modern web browser.
